# QuantJourney Public Plot Theme

The public backtester ships one report theme: `quantjourney`.

The commercial product keeps the broader report styling system private. The
public package only includes the pieces required to render consistent local
plots and the HTML dashboard generated by the example strategies.

### Key Files

#### date_formatter.py Provides date formatting functions for consistent date representation across plots. This file ensures uniform date formatting based on theme requirements.

#### manager.py Manages theme application and retrieval, with functionality to set, switch, and retrieve plot themes dynamically. It includes a main ThemeManager class, which standardizes theme usage within the QuantJourney framework.

#### types.py Defines theme-related types, including enumerations and constants used to specify plot elements. It provides type annotations and specifications that improve the readability and consistency of theme-related code.
